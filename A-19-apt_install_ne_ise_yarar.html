APT (Advanced Package Tool)


Kısaca APT olarak adlandırılan Advanced Package Tool, yani Gelişmiş Paketleme Aracı, Debian GNU/Linux ve Debian tabanlı Linux dağıtımlarının paket yönetim sistemidir. Apt ile yazılım kurma, yazılım kaldırma, sistemi güncelleme, çekirdeği derleme gibi işlemleri Uçbirim üzerinden gerçekleştirebilirsiniz.

Ubuntu gibi Debian temelli Linux dağıtımlarında, tanımlanmış kaynaklardan yeni yazılım paketlerini indirme ve bağımlılıkları ile birlikte kurma, paketleri yükseltme, paket liste içeriklerini güncelleme gibi işlemleri yerine getirmek için kullanılan bir komut satırı arayüzüdür.

Örnek uygulamalar:

Paket kurmak: sudo apt install paket_adı
Birden çok paket kurmak: sudo apt install paket1 paket2 paket3 paket4 paket5 paket6 ...
Kurulu bir paketi kaldırmak: sudo apt remove paket_adı
